## 🎥 Loom Video Recording

### Download Loom:
**Direct Link:** https://www.loom.com/download

### Or Use Online:
**Web Version:** https://www.loom.com/record

### Recording Steps:
1. Install/Open Loom
2. Click "New Video"
3. Select "Screen + Camera" or "Screen Only"
4. Click "Start Recording"
5. Run: `python run.py`
6. Open: `http://localhost:5000`
7. Login and demo features
8. Click "Stop Recording"
9. Share the link

## 🔧 If Port 5000 Busy

```bash

# Kill process (replace <PID> with actual number)
taskkill /PID <PID> /F
```
