# 🚀 NLPCRM: Next-Gen AI-Powered Customer Intelligence

(static/images/banner.png)

Welcome to **NLPCRM** – the intelligent Customer Relationship Management system that transforms unstructured conversations into actionable business intelligence. 

NLPCRM leverages state-of-the-art **Natural Language Processing (NLP)** to automatically extract contacts, company data, and lead interest from WhatsApp, Emails, Zoom, and Teams transcripts.

---

## ✨ Premium Features

### 🧠 Intelligent Extraction
- **AI-First Engine**: Powered by Hugging Face (Qwen 2.5), converting raw text into structured JSON entities.
- **Multichannel Sync**: Seamlessly ingests data from WhatsApp Web, Gmail, and Meeting Transcripts.
- **Auto-Pipeline**: Automatically categorizes leads into a visual sales pipeline based on AI-detected interest levels.

### 🛡️ Enterprise-Grade Security
- **Full User Management**: Secure registration, login, and password management.
- **Google OAuth**: One-click professional login with Google Account integration.
- **Advanced Protection**: CSRF protection, secure headers (Talisman), and rate-limiting (Flask-Limiter) built-in.

### 💎 Stunning Visual Experience
- **Glassmorphic UI**: High-end modern design with blur effects and sleek animations.
- **Live Activity Feed**: Real-time tracking of interactions and system events.
- **Installable PWA**: Experience the speed of a native app on any device.

---

## 🏗️ System Architecture

![Architecture](static/images/architecture.png)

Built with a robust micro-service architecture:
- **Backend**: Python 3.8+ / Flask / SQLite Cloud
- **NLP Layer**: Hugging Face Inference API / Transformers
- **Frontend**: Vanilla CSS (Premium Tokens) / Modern JS / Jinja2
- **Auth**: Google OAuth 2.0 / Werkzeug Security

Create a `.env` file with your credentials:
```env

HF_API_KEY=your_huggingface_key
GOOGLE_CLIENT_ID=your_google_id
GOOGLE_CLIENT_SECRET=your_google_secret
ADMIN_PASSWORD=admin@2026
```

### 3. Run the App
```powershell
python run.py
```
